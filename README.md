# hotel-booking-system
A simple program to manage hotel bookings in Rust

# Rust
Jeg har valgt at kode mit booking system i Rust, fordi at Rust er et hurtigt og sikkert sprog at skrive i. Python var også i overvejelserne på grund af dets simplicitet og support, men jeg endte med Rust, fordi jeg virkelig nyder at skrive i det.

# SQLite
Jeg har valgt at anvende SQLite til min database, for at holde styr på mine bookninger. SQLite er blevet valgt, fordi at det har god support med Rust, og det skaber en lokal database i en .db-fil.
